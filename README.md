# Tic Tac Toe

This is a **Tic Tac Toe game** built with **vanilla JavaScript, HTML, and CSS**, following the **Odin Project curriculum**. The game allows two players to play on the same device, tracks turns, and displays the winner or a draw.

---

## Features

- 3x3 grid Tic Tac Toe board
- Two players: X and O
- Clickable cells to place tokens
- Detects **winner** or **draw**
- Displays the winner or draw message below the board
- Board resets after game ends
- Responsive design and modern CSS styling

---

## Technologies Used

- **JavaScript** – game logic, player turns, and event handling
- **HTML5** – structure of the page
- **CSS3** – modern layout, grid system, hover and click effects, animations
